# Лабораторна робота №5: Хмарні технології

### Опис проєкту
Цей проєкт присвячений вивченню основ роботи з системою контролю версій GitHub. Він демонструє навички структурування репозиторію, використання гілок (branching), створення запитів на злиття (pull requests) та налаштування базової автоматизації (CI/CD).

**Мета:** Навчитися працювати з GitHub, організовувати структуру файлів, використовувати гілки та налаштовувати GitHub Actions.

### Інструкція запуску:
1. Відкрити термінал.
2. Запустити команду: `python3 src/app.py`
